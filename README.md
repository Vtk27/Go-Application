# Todo List App

A simple Todo List application built using Go.

## Features

* Add new tasks
* View all tasks
* Delete tasks
* Update tasks

## Usage

1. Run the application using `go run main.go`
2. Open a web browser and navigate to `http://localhost:8080`

## API Endpoints

* `GET /showtasks` - View all tasks
* `POST /addtask` - Add a new task
* `DELETE /deletetask?id=<task_id>` - Delete a task
* `PUT /updatetask` - Update a task
